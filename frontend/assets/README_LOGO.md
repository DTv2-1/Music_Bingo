# Perfect DJ Logo

Place the Perfect DJ logo file here with the name: `perfectdj_logo.png`

## Requirements:
- Format: PNG (with transparent background recommended)
- Recommended size: 400x160 pixels (or similar 5:2 aspect ratio)
- Will be displayed at 20mm x 8mm on bingo cards

## Location:
The logo will appear in the center "FREE" cell of each bingo card, between the "FREE" text and the website URL.
