## âš ï¸ Troubleshooting

### "Module not found" error
```bash
pip install fpdf2 requests
```

### "pool.json not found"
```bash
cd backend
python generate_pool.py
```

### "ElevenLabs API error: 401"
- Check API key is correct in `game.js`
- Ensure no extra spaces
- Key should start with `sk_`

### No sound playing
1. Click anywhere on page first
2. Check laptop volume
3. Check browser console (F12) for errors

### Songs won't load
- Check internet connection (iTunes API needs internet)
- Verify `data/pool.json` exists
- Try refreshing the page
